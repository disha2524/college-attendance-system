# Student Adjustment & Smart Attendance Management System

## Overview
The Student Adjustment & Smart Attendance Management System is a web-based platform designed to improve classroom management by digitalizing attendance tracking and enabling lecture adjustments between teachers.

The system provides separate dashboards for students and teachers, ensuring that both roles have access to the features they need for effective academic management.

## Key Features
- Separate login portals for Students and Teachers
- Digital attendance tracking
- Late entry marking for students
- Student dashboard for viewing attendance and marks
- Teacher lecture adjustment system
- Simple and user-friendly interface

## System Roles

### Student Login
Students can log in to their personal dashboard where they can:

- View their attendance records
- Check their marks and academic performance
- Track late attendance entries
- Monitor updates related to their classes

This allows students to stay informed about their academic progress in real time.

### Teacher Login
Teachers have additional management functionalities in the system.

They can:
- Mark student attendance (Present / Absent / Late)
- Update attendance records
- Manage lecture sessions
- Handle lecture adjustments when a teacher is absent

### Late Entry Tracking
If a student arrives late to class, the teacher can mark the student as **Late** instead of marking them present or absent.  
This ensures more accurate attendance monitoring.

### Lecture Adjustment Feature
One of the main features of the system is the **teacher lecture adjustment mechanism**.

- If a teacher is unable to take a lecture, they can assign the class to another teacher.
- Teachers who are free during that time can voluntarily take the lecture.
- This prevents lecture cancellations and ensures continuous learning.

## Problem It Solves
Many institutions still manage attendance and lecture adjustments manually, which can lead to confusion and missed classes.  
This system helps automate these processes and improves overall classroom coordination.
